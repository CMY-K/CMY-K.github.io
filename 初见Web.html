初见Web

一位编程新手某天看到大佬们的精美个人博客，心生向往，于是决定自己也造一个……
但由于太菜，只能浏览无数经验贴，在与各种问题反反复复的斗争中孵化了这个blog。
无所谓，就让这个幼稚 blog 陪着本菜鸟一起成长吧

万事开头难

找了两篇博客开始学着搭建基于github的个人网站，但是困难重重。
在一次又一次的失败之后，我决定重头再来一遍，记录整个犯错的过程，便于复盘，抱着这样的心态我居然一次性成功了（惊喜

步骤

学习hexo和css基础知识 -> 安装 Node.js -> 下载 Git 和注册 GitHub -> 配置 GitHub SSH -> 安装 hexo -> 将 Hexo 发布到GitHub -> 配置主题Fluid -> 更改域名 -> 学习相关操作，按照个人想法完善博客

参考博客来源

搭建网页雏形和fluid主题的配置主要参考了一些优秀博客 and csdn，网上的教程非常多，内容也很全，步骤也很清楚，我在这里就不赘述了。
个人建议是跟着一篇走到黑，不然就会像我一样步骤乱序，各种终端指令搞不明白，导致一波混合操作之后出现各种环境bug，然后又要将自己亲手种下的 bug 一个个解决掉，可以说3/4 的时间都花费在解决这些 bug 上了。
不过也是这样反反复复折腾之后，对终端也更加熟悉了，也学到了一些基本命令行操作，过程虽然痛苦，但是收获也颇多。

主题配置

对比许多主题之后，决定选择fluid主题，页面排版简洁，排版更专注于内容而非一些花里胡哨的设计，风格高级，而且一些小设计（比如首页向上拖动的按钮）深得我心。
